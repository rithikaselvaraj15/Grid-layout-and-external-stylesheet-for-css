EXAMPLES OF USEING THOSE PROPERTIES

1. grid-template-columns & grid-template-rows

grid-template-columns- Sets how many columns and their width. Example: grid-template-columns: 100px 100px 100px; (3 columns of 100px each)
grid-template-row - Sets how many rows and their height. Example: grid-template-rows: 100px 200px;

3. grid-auto-flow

grid-auto-flow controls how the grid automatically places items that are not given a specific position (like grid-row or grid-column). It decides the direction in which the grid fills items.

4. gap (row-gap & column-gap)

gap is the property used to give space between grid items — like a margin inside the grid layout.
It doesn’t add space outside the grid, only between the boxes inside it. Before, people used grid-gap, but now we simply use gap.

5.justify-items & align-items

These two properties are used to control how items are positioned inside each grid cell (not the entire grid). Think of each grid box as a small container — justify-items and align-items decide where the content sits inside each box.
